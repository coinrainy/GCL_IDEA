# Experiment Plan: SIVA-GCL-positive-core

**Problem**：false-negative GCL needs a more innovative non-loss mechanism; ordinary positives may be pre-aligned after message passing.  
**Method Thesis**：Semantic stability constrained intervention positives create informative node-local positive views and reduce reliance on real-node hard negatives.  
**Date**：2026-06-26  
**Stage**：smoke planning only

## Claim Map

| Claim | Why It Matters | Minimum Convincing Evidence | Linked Blocks |
|---|---|---|---|
| C1: SIVA positives are nontrivial but semantically stable | differentiates from random augmentation | view distance high, stability/reconstruction acceptable | B1 |
| C2: SIVA increases positive learning signal after message passing | addresses SPGCL prealignment | post-GNN positive similarity and positive gradient norm more informative than GRACE/GCA | B2 |
| C3: SIVA is not just GraphMAE/critic regularization | protects novelty | GraphMAE-only/critic-only weaker than SIVA | B3 |
| C4: SIVA is not just stronger random perturbation | protects mechanism | random intervention matched by edit distance weaker than SIVA | B4 |

## Experiment Blocks

### Block 1: Critic and Intervention Sanity

- Dataset / split：Cora, `1:1:8`, seed 0。
- Systems：critic pretrain, SIVA-positive generator。
- Metrics：reconstruction loss, stability score, intervention edit distance, view distance。
- Success criterion：can generate positives with larger view distance than GRACE augmentation while retaining stability。
- Priority：MUST-RUN。

### Block 2: SIVA-positive Smoke

- Systems：GRACE, GCA-style adaptive augmentation, SIVA-positive core。
- Metrics：LogReg val/test@best, post-GNN positive similarity, positive gradient norm, uniformity/alignment。
- Success criterion：SIVA does not underperform GRACE badly and improves positive informativeness metrics。
- Priority：MUST-RUN。

### Block 3: Critic Controls

- Systems：GraphMAE-only / critic-only, critic-shuffled SIVA, SIVA-positive core。
- Metrics：LogReg, stability, positive informativeness。
- Success criterion：SIVA > critic-only and SIVA > critic-shuffled in mechanism metrics。
- Priority：MUST-RUN。

### Block 4: Random Intervention Control

- Systems：random intervention positive matched by edit distance, SIVA-positive core。
- Metrics：LogReg, stability, positive gradient norm。
- Success criterion：SIVA > random intervention; otherwise semantic critic narrative fails。
- Priority：MUST-RUN。

### Block 5: Virtual Negative Late Ablation

- Systems：SIVA-positive core, SIVA-positive + virtual-negative。
- Purpose：risk observation only, not core claim。
- Kill rule：delete virtual-negative if label-only false-negative exposure is not lower than real hard negatives。
- Priority：NICE-TO-HAVE after B1-B4 pass。

## Kill Rules

- If GraphMAE-only or critic-only matches SIVA, kill SIVA as a distinct GCL idea。
- If random intervention matches SIVA, kill semantic stability critic narrative。
- If SIVA-positive view distance is not higher than GRACE/GCA, kill intervention-positive。
- If stability collapses, revise intervention constraints before any pilot。
- If post-GNN positive similarity / positive gradient is not more informative than GRACE, kill anti-prealignment claim。
- If smoke requires much larger budget with no mechanism metric gain, pivot。
