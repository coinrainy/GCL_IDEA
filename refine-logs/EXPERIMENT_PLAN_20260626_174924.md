## Hard Kill Rules

- `G8 <= max(G0,G1,G2,G6,G7)`：kill。
- `G3` or `G9` within 0.3 absolute accuracy points of G8：kill as random/task artifact。
- `G8 <= max(G4,G5)`：kill as no synergy beyond single-factor prior。
- `G8` only wins by using larger params, more epochs, or more task search：kill as unfair budget。
- G8 improves task diagnostics but not frozen LogReg accuracy over matched controls：kill。
- Any task generator reads true labels, validation accuracy, or test labels：BLOCKED/invalid。
