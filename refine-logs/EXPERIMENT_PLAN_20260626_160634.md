## Smoke ID

`PA-CER-M0-001`

## Research Question

Can train-label logistic probes route anchors between instance discrimination and masked readout-preserving consistency better than pair-level false-negative repair or representation diagnostic regularization?

## Systems

| ID | System | Description | Required |
|---|---|---|---|
| P0 | GRACE | baseline under same split/evaluator | yes |
| P1 | probe-only diagnostic | warmup + train probe, no encoder update | yes |
| P2 | mask-only consistency | masked consistency without probe routing | yes |
| P3 | PACER full | probe-calibrated objective routing | yes |
| P4 | shuffled-label probe | same pipeline with shuffled train labels | yes |
| P5 | random-probe route | random anchor route with matched route fraction | yes |
| P6 | scalar reweight control | use q_i as scalar weight, no objective type switch | yes |
| P7 | train-only CV probe | internal train-fold probe, no val labels in calibration | should |
| P8 | supervised-contrastive small-label control | compare against a straightforward labeled-node SupCon/semi-supervised control | should |
