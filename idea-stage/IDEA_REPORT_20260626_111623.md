# Idea Discovery Report: SIVA-GCL-positive-core

**方向**：图对比学习假负样本方向，面向普通节点分类，要求避免 loss-only trick，追求更有创新性的 2026 论文机制  
**日期**：2026-06-26  
**Pipeline**：research-lit → idea-creator → novelty-check → research-review → research-refine-pipeline  
**阶段结论**：`REVISE_TO_SIVA_POSITIVE_SMOKE_PLANNING`

## Executive Summary

用户指出 BOND-GCL “只是在 loss 上做文章”，该判断成立。BOND 已降级为 loss-side baseline / ablation，不再作为当前主线。本轮新的推荐方向是 **SIVA-GCL-positive-core / Semantic Stability Critic constrained Semantic-Preserving Intervention Positive**。

SIVA 的主贡献不是生成 hard negatives，也不是改 InfoNCE loss，而是为每个 anchor 搜索 **尽可能不同但语义稳定的 node-local intervention positive**。这直接回应 SPGCL 2026 提出的 positive pre-alignment 问题：普通 GCL 的 positive pair 在 message passing 后可能已经过度相似，导致正样本学习信号弱；SIVA 试图主动制造更有信息量的 positives，从源头减少对真实节点 hard negatives 的依赖。

fresh `gcl_scientific_reviewer` 给出 `REVISE`、novelty `6.7/10`、confidence `0.70`。结论：SIVA 比 BOND 更有论文形态，但必须收缩为 `semantic stability critic + intervention-positive search`，virtual-negative 只能作为 late ablation。

## Literature Landscape

当前 false-negative / graph SSL 相关 prior 已经覆盖：

- CGC：counterfactual hard negatives。
- BalanceGCL：graph-level balanced hard negatives + semantic positives。
- RGCL/GCA/ACGA：learned / rationale-aware / adversarial graph augmentation。
- DoG：graph diffusion 生成 synthetic graph structures，用于 node classification 和 GCL。
- SPGCL 2026：message passing 导致 positive pre-alignment，需恢复 positive learning signal。
- GraphMAE / masked graph modeling：masked reconstruction 可做 generative self-supervision。

因此新方法不能主打“生成 hard negatives”或“learned augmentation”。SIVA 的差异化应落在：**critic-constrained maximal semantic-preserving intervention positive**。

## Recommended Idea

### Idea 1: SIVA-GCL-positive-core

- **Method (what we actually do)**：训练轻量 masked-context semantic stability critic；对每个 anchor 的 ego-context 搜索 feature/edge/neighbor intervention；选择 view distance 最大但 critic 仍判定语义稳定的 positive view；用它替代普通随机增强 positive 进入 GRACE/GCA 训练框架。
- **Hypothesis**：比随机增强更远、但语义稳定的 positives 能缓解 message-passing pre-alignment，使 positive gradient 更有信息量，并降低模型对真实节点 hard negatives 的依赖。
- **Minimum experiment**：Cora seed=0 smoke；对照 GRACE、GCA-style aug、GraphMAE-only/critic-only、random intervention positive、SIVA-positive core。
- **Expected outcome**：SIVA-positive 的 view distance 高于 GRACE/GCA，semantic stability 不崩，post-GNN positive similarity / positive gradient 更 informative，LogReg 不低于 GRACE。
- **Novelty**：6.7/10；closest prior 是 SPGCL、RGCL、GraphMAE、CGC、BalanceGCL、ACGA、DoG。
- **Feasibility**：中等；首版用 greedy / differentiable mask search，不训练复杂 generator。
- **Risk**：MEDIUM-HIGH；critic 可能学到 identity/degree shortcut，或 random intervention 就能匹配 SIVA。
- **Contribution type**：view generation / intervention mechanism。
- **Pilot result**：SKIPPED；本轮未跑新实验。
- **Reviewer likely objection**：SIVA 可能只是 GraphMAE + GCA/RGCL 拼装；必须用 critic-shuffled、GraphMAE-only、random intervention 对照证明 intervention-positive 不可替代。
