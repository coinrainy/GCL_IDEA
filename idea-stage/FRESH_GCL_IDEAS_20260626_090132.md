## 近期 prior work 边界

- ProGCL 已明确指出图上相似 hard negative 经常是假负，并估计 true-negative probability。
- KDD 2024 Node Similarity 已覆盖“理想目标、扩正样本、去假负、节点相似度建模”。
- SIGIR 2025 IFL-GCL 已把 GCL 解释成 Positive-Unlabeled learning，并用 InfoNCE 表示相似度重采样语义样本。
- IJCAI 2025 NML-GCL 已用 learnable negative metric network 和 bi-level training 学 soft negative status。
- 2025/2026 的 hard negative、uncertainty、semantic-aware positive 方向已经拥挤，普通 weighted negative 或 hard-negative mining 不宜作为主贡献。

## Top 3

1. **Ambiguity-Set Soft Target InfoNCE**  
   排名第一，因为它在拥挤的 false-negative 方向中仍有清楚的可写 delta：从点式 pseudo-label/soft weight 变为 set-valued pair semantics。工程上只需改 GRACE/GCA loss，失败也容易诊断。

2. **Semantics-Preserving Augmentation Auditor**  
   排名第二，因为它绕开了“再做一个 negative reweighting”的拥挤路线，直接攻击 GRACE/GCA 视图语义不保真的根因。当前仓库已有 GRACE/GCA 复现，适合做最小替换。

3. **Pair Gradient Budget GCL**  
   排名第三，因为它有明确诊断价值：如果 repulsive gradient mass 确实来自疑似假负或 dense community，限制预算应能减少类内撕裂；如果失败，也能产出很干净的负结果。

## Top 1 Anchor

**Problem Anchor**: 节点级 GCL 中，高相似非增强节点既可能是同类候选正样本，也可能是真 hard negative；现有方法常把这种不确定性压成一个点估计，从而在过度转正和假负推远之间摇摆。

**Method Thesis**: 用 pair-level positive-probability interval 取代点式 pseudo-positive，将高置信正样本拉近、低 upper-bound 节点作为可靠 hard negatives，而对 ambiguous pairs 保持保守，从而让 InfoNCE 的吸引和排斥都受不确定性约束。

## Interval / Ambiguity-Set Soft Target InfoNCE Assessment

判断：**值得作为 top1 进入 refinement，但 novelty 必须写成 set-valued / interval-valued pair semantics，而不是普通 soft target InfoNCE 或 similarity weighting。**

相对 IFL-GCL：IFL-GCL 的核心是 PU 视角和用 InfoNCE 表示相似度提取语义后重采样；interval 方案的 delta 是承认 posterior 不可精确识别，只给出 lower/upper bounds，并设置 abstention 区，不把所有相似节点硬转正。

相对 Node Similarity：Node Similarity 已经把节点相似度用于扩正和去假负；interval 方案必须强调 epistemic uncertainty、区间校准和 worst-case/conservative target，而不是再提出一个相似度函数。

相对 NML-GCL：NML-GCL 学一个 negative metric network 并做 bi-level optimization；interval 方案应保持轻量、非参数或半参数，用稳定性/多信号一致性给出 bounds，不引入重 metric learner。

最小可写贡献应包括三件事：

1. 区间构造：`[l_ij,u_ij]` 来自 feature/structure/embedding/temporal stability 的保守组合。
2. 区间 InfoNCE：ambiguous pairs 不被强行转正，也不被当普通负样本强推远；hard negative 只来自 `u_ij < epsilon`。
3. 证伪 ablation：point target、only downweight、only positive expansion、no abstention、full interval 五组必须同协议比较。

主要风险：已有 soft target InfoNCE、Bayesian contrastive、DRO/ambiguity-set InfoNCE 等通用工作会削弱“区间 loss”本身的新颖性。因此图论文的核心不应是“我用了 soft target”，而应是“图节点 pair semantics 不可识别时，用区间和 abstention 防止相似节点被过度转正或假负推远”。
