## Executive Summary

本轮基于 ORBIT negative smoke 选择 **GRAFT-GCL / Graph Prior-Task Fitted Contrastive Learning** 作为新的机制级 pivot。ORBIT full `82.98` 低于 GRACE `84.78` 和 matched GRACE `84.96`，说明 operator-response basis target 虽有自监督信号，但不足以支持继续小调。

GRAFT 的核心转向是：不再优化节点对、masked evidence、operator response、rank/boundary diagnostic 或 train-label probe route，而是在同一个真实图上生成一族无标签的 plausible node-classification tasks，并用这些 synthetic task labels 训练 task-conditioned supervised contrastive representation。真实标签只在最终 frozen Logistic Regression evaluator 中使用。

## Literature Boundary

- GraphPFN：使用合成 attributed graph/task prior 训练 graph foundation model。GRAFT 不做跨图 foundation/in-context 模型，只在当前真实图上做 label-free task prior GCL。Source: https://arxiv.org/abs/2509.21489
- AutoSSL：自动组合多个 graph SSL pretext tasks。GRAFT 不搜索 pretext 权重，而是定义 node-classification task prior。Source: https://arxiv.org/abs/2106.05470
- AGSSL：多 teacher pretext knowledge distillation。GRAFT 不蒸馏 teacher，也不继续 ORBIT response teacher。Source: https://arxiv.org/abs/2210.02099
- PCL：半监督 pseudo-label / negative-pair contrastive learning。GRAFT 不使用 train-label pseudo-labeler，不为真实任务造伪标签。Source: https://arxiv.org/html/2302.09532v3

## Method Thesis

> 少标签节点分类真正需要的是一个对“可能的节点分类任务族”有良好线性可分性的 encoder，而不是对某个单一 pretext target 的高拟合。

GRAFT 将 graph SSL objective 改成 prior-task fitting：

1. 在真实图 `G=(A,X)` 上构造无标签 task generator `p(tau | G)`。
2. 每个 task `tau` 产生 synthetic class assignment `y_i^tau`，代表一种 plausible node-classification boundary。
3. Encoder `f_theta` 对每个 task 使用 task-conditioned supervised contrastive loss：

```text
L_GRAFT = E_{tau ~ p(tau|G)} SupCon({f_theta(v_i)}, {y_i^tau})
```

4. 通过 task dropout、task family balancing 和 collapse guard 避免只记住单一 clustering。
5. 训练结束后冻结 encoder，用项目统一 Logistic Regression evaluator 在真实 train/val/test split 上评估。

## Task Prior Families

最小 GRAFT 只允许四类预注册任务：

1. **Feature mixture tasks**：从稀疏 feature 子空间采样 class boundary，再做轻量 graph smoothing。
2. **Topology role tasks**：基于 degree、ego density、PageRank/diffusion bucket、Laplacian positional bins 产生 role labels。
3. **Homophily-controlled MRF tasks**：用 Potts/Ising-style label smoothing 采样不同 edge-label correlation 的任务。
4. **Feature-topology interaction tasks**：让 labels 同时依赖 feature residual 与 topology role，避免只学 feature 或只学 structure。

禁止任务生成器读取真实 label、validation accuracy 或 test performance。

## Paper Skeleton

### Title

Graph Prior-Task Fitted Contrastive Learning for Node Classification

### Abstract Thesis

Graph contrastive learning often optimizes view agreement, masked reconstruction, or hand-crafted predictive targets. We argue that these objectives are weak proxies for label-scarce node classification. GRAFT instead samples a distribution of label-free but plausible node-classification tasks on the observed graph and fits an encoder by task-conditioned contrastive meta-risk. This trains representations for downstream task transfer without using real labels during pretraining.

### Contributions

1. 提出 same-graph prior-task fitting 作为 graph contrastive learning 的新 supervision object。
2. 设计 label-free node-classification task generator，覆盖 feature、topology、homophily 和 interaction boundaries。
3. 给出 task-conditioned SupCon meta-risk 与 task-prior balance / dropout 机制。
4. 提供严格 controls，区分 GRAFT 与 pseudo-labeling、AutoSSL、GraphPFN、PPR/clustering 和 ORBIT response learning。
