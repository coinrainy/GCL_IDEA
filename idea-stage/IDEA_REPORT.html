Executive Summary

本轮重新构思选择 ORBIT-GCL / Operator-Response Basis Induction for Graph Contrastive Learning 作为当前 Top1。它不再继续 false-negative pair mining、positive/neutral closure、rank/boundary diagnostic、train-label probe routing 或 masked evidence group prediction，而是把自监督目标改成：学习每个节点在一组无标签 graph intervention operators 下的 response field。

Fresh gcl_scientific_reviewer 给出 REVISE，confidence 0.73。ORBIT 是当前候选中最像完整论文主线的方向，但必须先收缩为“node-level operator response field 的低秩 response-basis prediction”，并通过 Cora seed0 kill-smoke 证明它不是 GraphMAE/GraphMAE2、Graph-JEPA、TTER/D-SLA、GCIL/AS-GCL 或 regularization/capacity 的换名。

Literature Boundary

ORBIT 必须与以下近邻切开：

  • GRACE：node-level contrastive agreement with graph corruption。参考：https://arxiv.org/abs/2006.04131
  • GraphMAE / GraphMAE2：masked feature reconstruction、remask decoding、latent prediction。参考：https://arxiv.org/abs/2205.10803 和 https://arxiv.org/abs/2304.04779
  • GraphPAE / StructPosGSSL：结构与位置重建或结构/位置感知 SSL。参考：https://arxiv.org/html/2505.23345v2 和 https://arxiv.org/html/2502.16233v1
  • GCIL / AS-GCL：causal/spectral invariance 与 spectral augmentation。参考：https://arxiv.org/html/2401.12564 和 https://arxiv.org/html/2502.13525v1
  • Graph-JEPA / PCR：context-target latent prediction。参考：https://arxiv.org/html/2502.01684v4

ORBIT 的差异点不能写成“更多 augmentation”或“latent prediction”，而必须写成：

operator-valued node response target：一个节点在多个干预算子下的响应函数，而不是输入、节点身份、operator id 或单个 latent target。

Ranked Ideas

1. ORBIT-GCL / Operator-Response Basis Induction

核心 thesis：少标签节点分类需要 class-readable perturbation sensitivity。ORBIT 用固定无标签 operator bank 产生节点 response field，再训练 encoder 从 clean / weak view 预测 response basis，使表示显式携带“哪些图证据被扰动时该节点语义响应会改变”的信息。

最小机制：

  1. 固定 operator bank Omega = {O_m}，包括 feature-frequency mask、degree-preserving edge perturbation、diffusion-temperature shift、ego-radius crop、positional perturbation。
  2. EMA teacher 产生 response field：
R_i = [h_T(O_1(G), i) - h_T(G, i), ..., h_T(O_M(G), i) - h_T(G, i)]
  1. 对 R_i 做 train-label-free low-rank basis projection，得到 b_i。
  2. Student 从 clean / weak view 表示预测 b_i，并在 held-out operators 上评估 response generalization。
  3. Frozen encoder + Logistic Regression downstream，沿用项目统一 split/evaluator。

为什么像完整论文主线：

  • Problem：Graph SSL objective mismatch for label-scarce node classification。
  • Method：operator response basis induction。
  • Theory/analysis：response basis 与 class-readable perturbation sensitivity 的关系。
  • Experiments：强对照覆盖 GCL、masked SSL、JEPA、transformation prediction、discrepancy regression、response controls 和 shortcut controls。
  • Ablation：operator family、basis rank、teacher source、held-out operator generalization、shortcut removal。

Reviewer verdict：REVISE。保留，但不能直接 GO。

2. MORPH-GCL / Morphological Ego Program Learning

用 ego graph edit program 作为自监督目标，训练 encoder 预测局部图形态程序 outcome / edit distance。创新性可能较强，但离普通节点分类 GCL 太远，且容易撞 Graph Edit Networks / GEDGNN / topology transformation SSL。状态：PIVOT，仅作备选。

3. SCALE-GCL / Multi-scale Evidence Conservation

用 node/ego/community/global evidence flow 的 conservation law 约束表示。该方向容易重复 SPECTRA：诊断改善但 accuracy 不转化。状态：KILL/weak PIVOT。

4. ATLAS-GCL / Self-Supervised Label-Function Atlas

训练多族 label-free pseudo-label functions，构建 latent task atlas。该方向太像 AutoSSL / multi-pretext stacking。状态：KILL。

Refined ORBIT Proposal

Problem Anchor

普通节点分类中的 Graph SSL 不是缺少更多 pair heuristics，而是缺少对“节点响应函数”的建模。一个节点的类别可读证据往往表现为对特征频段、结构扰动、扩散尺度、ego radius 和位置扰动的敏感性组合。现有 node-node contrastive / masked reconstruction / latent target prediction 没有把这种 operator response 作为主要学习对象。

Method Thesis

ORBIT 将 graph SSL 的自监督目标改为 node-level operator response basis prediction：让 encoder 在不使用标签、不使用 test 信息、不构造 cross-node positives 的情况下学习每个节点在多种图干预下的响应模式。

Non-Goals

  • 不做 kNN/PPR/CAST mining。
  • 不做 cross-node positive/neutral closure。
  • 不做 train-label probe routing。
  • 不做 masked feature/edge/position reconstruction 作为主目标。
  • 不做多 pretext stacking。
  • 不把 Barlow/VICReg/covariance guard 作为贡献；如使用，只能作为 collapse control 并做 no-guard ablation。

Paper Skeleton

Title

Operator-Response Basis Induction for Graph Contrastive Learning

Abstract thesis

Graph SSL for node classification usually learns invariance across augmented views or reconstructs masked inputs. We argue that label-scarce node classification requires a different target: the response function of each node under graph interventions. ORBIT induces low-rank node-level response bases from a fixed bank of label-free graph operators and trains an encoder to predict these bases from clean views. This replaces node-node negatives and masked reconstruction with operator-valued self-supervision.

Main contributions

  1. 提出 operator response field 作为 graph SSL 的新 supervision object。
  2. 给出 ORBIT 的无标签 response-basis induction 方法。
  3. 设计 held-out operator generalization 与 shortcut controls，用于区分 response learning 与 transformation-id / degree-feature shortcut。
  4. 在节点分类协议下与 GCL、masked SSL、JEPA、transformation prediction、discrepancy regression 强对照比较。

Expected sections

  1. Introduction：为什么 false-negative 修补和 masked evidence 都不够。
  2. Related Work：GCL、masked graph autoencoders、predictive graph SSL、causal/spectral graph SSL、transformation/discrepancy SSL。
  3. Problem Formulation：operator response field。
  4. Method：operator bank、teacher response field、basis projection、student objective、held-out operator test。
  5. Experiments：protocol、baselines、main results、ablation、diagnostics、failure analysis。
  6. Limitations：operator bank dependence、teacher bias、cost、transductive assumptions。
