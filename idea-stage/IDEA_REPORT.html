Selected Top1: PACER-GCL

One-sentence thesis：PACER-GCL uses train-label logistic probes to estimate which anchors are fragile for the downstream linear readout, then changes their SSL objective type from instance discrimination to masked logit-preserving consistency, avoiding pair mining while aligning pretraining with the actual node-classification evaluator.

Core Mechanism

PACER has three stages:

  1. Warmup：train a normal GRACE encoder for a short fixed budget。
  2. Probe calibration：freeze warmup embeddings and train a Logistic Regression probe using train labels only。Validation labels can select fixed hyperparameters or early stopping, but cannot train the probe objective or thresholds used on test。

  3. Objective routing：

    • readout-stable anchors：continue normal GRACE objective；
    • readout-fragile anchors：switch to masked logit-preserving consistency。Two masked views must preserve the train-probe logit direction/margin instead of being pushed through all-node instance discrimination。

Important：PACER does not select cross-node positives, does not smooth relations, and does not score unlabeled node pairs as same-class.

Why It Addresses False Negatives

False negatives hurt because instance discrimination may optimize away information that a small linear classifier needs. PACER does not try to identify false-negative pairs. It detects anchors for which the current representation is fragile under the downstream readout and changes the pretraining treatment for those anchors. This reframes false negatives as an objective-treatment mismatch problem rather than a pair-classification problem.

Novelty Boundary

PACER must not claim novelty for:

  • supervised contrastive learning；
  • semi-supervised GCL；
  • simple probe loss；
  • validation-based model selection；
  • sample reweighting。

The defensible boundary is narrower:

evaluator-aligned SSL objective routing for node-level GCL, where train-label probes decide objective type rather than pair labels, thresholds, relation smoothing, or scalar weights.
