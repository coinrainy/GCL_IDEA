| Prior line | 已覆盖内容 | 对 IGT-GCL 的约束 |

## 5. Recommended Idea: IGT-GCL

### Problem Anchor

节点级 GCL 中，高相似非增强节点既可能是同类 candidate positive，也可能是真 hard negative。现有方法经常把这种不确定性压成一个点式 pseudo-label 或点式权重，导致两类错误：

1. 过度转正：把语义不同但相似的节点拉近。
2. 假负推远：把真实同类或同语义节点作为 hard negative 强推远。

### Method Thesis

IGT-GCL 不直接估计单点 `p(y_i = y_j)`。它为 pair 构造区间 `[l_ij, u_ij]`：

- `l_ij`：保守 positive lower bound。只有多个信号一致支持时才高。
- `u_ij`：宽松 positive upper bound。只要任何可靠信号认为可能同类，就保持较高。
- `l_ij` 高：作为 conservative soft positive。
- `u_ij` 低且当前 embedding similarity 高：作为 reliable hard true negative。
- `l_ij` 低、`u_ij` 高：作为 ambiguous pair，执行 abstention 或 stop-gradient denominator。

该方法只替换 contrastive target construction / pair mask，不改变 encoder、split、evaluator 或正式评估协议。

### Minimal Mechanism

1. Warmup：用 vanilla GRACE/GCA 训练少量 epoch，并维护 EMA encoder。
2. Pair signals：
   - feature similarity rank；
   - structure proximity rank，例如 PPR / shared-neighbor / k-hop score；
   - EMA embedding similarity rank；
   - multi-augmentation / multi-checkpoint rank stability。
3. Interval construction：
   - lower bound 使用 conservative agreement，例如多个信号的低分位或 minimum；
   - upper bound 使用 liberal union，例如多个信号的高分位或 maximum；
   - pilot 必须验证它不是任意两个阈值，而是能产生有意义 coverage / precision 的 interval。
4. Interval-guarded InfoNCE：
   - self-positive 始终保留；
   - high-`l` pair 以小权重进入 numerator；
   - low-`u` and embedding-hard pair 进入加权 denominator；
   - ambiguous pair 不产生强 repulsive gradient。

## 8. Minimum Evidence Before Any Claim

第一阶段只允许 smoke/pilot，不允许 formal claim。必须先观察：

- `high-l` pair 的 same-label precision 是否高于 point posterior top-k。
- `low-u` hard pair 的 true-negative precision 是否高于普通 hard negatives。
- ambiguous set 是否富集 false negatives，而不是随机噪声。
- full IGT-GCL 是否优于 midpoint posterior、only-downweight、only-expansion。
- alignment/uniformity、embedding rank、class-wise/degree-wise 表现是否没有明显 collapse。

标签只能用于离线诊断，不得进入 unsupervised pair construction 或 test-time 选择。

## 9. Backup Pivot Rules

若出现以下情况，应停止主推 IGT-GCL：

- full interval 不优于 midpoint posterior：pivot 到 stability-calibrated point target 或 diagnostic paper。
- only downweight 匹配 full method：承认收益来自 denominator shrinkage，转向 Pair Gradient Budget GCL。
- only positive expansion 匹配 full method：转向 conservative node similarity / positive scarcity 方向。
- ambiguous abstention 导致 uniformity/rank collapse：保留为分析工具，不作为主训练 objective。
- Amazon/heterophily fixed split 明显恶化：不能写 general false-negative solution，只能写 homophily-bound 方法或停止。
